## 🎯 Fluxo Recomendado para Novos Desenvolvedores

### 1. Setup Inicial (5-10 minutos)
```bash
# Clone o repositório
git clone [repo-url]
cd VeloFlux

# Execute o setup rápido
./scripts/setup/install.sh

# Verifique o status
./scripts/utils/check-status.sh
```
